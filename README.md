# LY.WebApi

#### 介绍
基于.Net 6实现的WebApi框架。
IOC运用AutoFac+反射，实现特性注入。
避免了构造函数注入的臃肿，注入更优雅，生命周期控制更灵活。

ORM：SQLSugar
接口文档：Swagger
身份认证：JWT

#### 软件架构
-LY.Common
-LY.IServices
-LY.Model
-LY.Repository
-LY.Services
-LY.WebApi
